# Clean System Rebuild Checklist

## Step 1 – OS Installation
- Install Windows
- Apply latest updates
- Set hostname & domain settings

## Step 2 – Driver Installation
- Install chipset drivers
- Install network drivers
- Install robotic hardware drivers

## Step 3 – Middleware & Control Software
- Install control software
- Configure service dependencies
- Validate hardware integration

## Step 4 – Azure Integration
- Configure network
- Validate DNS
- Run connectivity test script

## Final Validation
- End-to-end data flow confirmed
- Full documentation updated
