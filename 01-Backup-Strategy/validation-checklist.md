# Backup & Restore Validation Checklist

## Pre-Restore
- Confirm backup integrity
- Verify image file checksum

## Post-Restore Validation

- OS boots without error
- Device Manager shows no driver issues
- Robotic hardware recognized
- Network connectivity functional
- Azure endpoint reachable
- Control software operational

## Final Signoff

Restore successfully replicates original system state.
